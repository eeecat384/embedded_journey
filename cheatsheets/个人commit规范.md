## 基本格式

```plain
<type>(可选:<scope>): <动作+内容简述>

[可选：学到了什么/解决了什么问题/关键参数]
```

### Type

| 类型        | 含义        | 使用场景                                        |
| ----------- | ----------- | ----------------------------------------------- |
| **`feat`**  | 新技能/代码 | 刚学会PWM、完成了一段新代码、掌握了某个知识点   |
| **`fix`**   | 修正错误    | 改bug、纠正之前的理解错误、电路返修             |
| **`note`**  | 笔记整理    | 往notes/里添加学习笔记、docs/添加参考文章       |
| **`proj`**  | 项目进展    | project/文件夹的里程碑（完成某个模块/打板成功） |
| **`chore`** | 杂项维护    | 整理文件夹、更新脚本、修改README                |

### Scope

| Scope               | 对应内容                         |
| ------------------- | -------------------------------- |
| **`embedded`**      | STM32裸机代码、驱动、外设学习    |
| **`linux`**         | 嵌入式Linux、驱动、应用层        |
| **`hw`**            | 硬件相关（立创EDA、焊接、电路）  |
| **`cpp`** / **`c`** | C语言学习笔记和代码              |
| **`git`**           | Git操作心得、协作流程            |
| **`cheat`**         | cheatsheets更新                  |
| **`repo`**          | 整个仓库级别（README、结构改动） |
