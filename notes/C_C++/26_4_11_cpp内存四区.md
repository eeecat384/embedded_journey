# cpp内存四区：代码区，全局区，堆区，栈区

不同区的代码拥有不同的生命周期，能有更大的灵活性和效率。

---

## 代码区

存储程序的代码，为编译后的机器码。

- 代码区是**只读**的，防止程序被恶意修改
- 代码区是**共享**的，对于频繁执行的程序，只存放一份，节省内存资源

---

## 全局区

存储全局变量和静态变量。

- 全局区也包括了**常量区**，字符串常量和其他常量也在这里存储
- 程序结束后会自动回收

---

## 栈区

存储局部变量和函数调用信息。

- 编译器自动分配并释放

---

## 堆区

存储动态分配的内存。

- 需要程序员手动分配和释放，结束时由程序释放
- C中用 `malloc` / `free`，C++中用 `new` / `delete`

---

## 变量类型与存储位置对照表

| 变量类型                     | 存储位置                   | 生命周期       |
| ---------------------------- | -------------------------- | -------------- |
| 全局变量（main外）           | 全局区                     | 程序运行期     |
| 静态全局变量（static）       | 全局区                     | 程序运行期     |
| 静态局部变量（main内static） | 全局区                     | 程序运行期     |
| 局部变量                     | 栈区                       | 函数执行期间   |
| 函数参数                     | 栈区                       | 函数执行期间   |
| new/malloc分配的内存         | 堆区                       | 手动释放前     |
| 字符串常量                   | 常量区                     | 程序运行期     |
| const常量                    | 常量区（或编译期优化）     | 程序运行期     |
| 类成员变量                   | 随对象存储（栈/堆/全局区） | 随对象生命周期 |

## 代码示例
```cpp
// ========== 全局区变量 ==========

int g_var = 10;                    // 全局区（已初始化数据段）
int g_var2;                        // 全局区（未初始化BSS段，自动置0）
static int s_g_var = 20;           // 全局区（静态全局变量）

const int c_g_var = 100;           // 常量区（全局const常量）

// ========== 类定义 ==========

class MyClass {
    int member;                    // 随对象存储：对象在栈则在栈区，对象在堆则在堆区
    static int s_member;           // 全局区（静态成员变量，类外初始化）
};

int MyClass::s_member = 0;         // 全局区

// ========== 函数示例 ==========

void example(int param) {          // param: 栈区（函数参数）

    // ----- 栈区变量 -----
    int local = 5;                 // 栈区（局部变量）
    int arr[10];                   // 栈区（局部数组）
    int* ptr = &local;             // ptr: 栈区（局部指针变量）

    // ----- 堆区变量 -----
    int* p = new int(10);          // p: 栈区（指针本身）
                                   // *p: 堆区（new分配的内存）
    int* arr2 = new int[5];        // arr2: 栈区
                                   // arr2指向的数组: 堆区

    // ----- 常量区变量 -----
    const char* str = "hello";     // str: 栈区（指针本身）
                                   // "hello": 常量区（字符串常量）
    const char* str2 = "hello";    // str2: 栈区
                                   // "hello": 常量区（与上面同一个地址，复用）

    // ----- const变量（分情况）-----
    const int c_local = 100;       // 栈区（局部const，编译器可能优化为立即数）
    static const int sc_local = 200; // 常量区（静态const）

    // ----- 静态局部变量 -----
    static int s_local = 30;       // 全局区（静态局部变量，只初始化一次，值保留）

    // ----- 对象成员变量 -----
    MyClass obj;                   // obj: 栈区（对象本身）
                                   // obj.member: 栈区（随对象在栈区）

    MyClass* pObj = new MyClass(); // pObj: 栈区（指针本身）
                                   // pObj->member: 堆区（随对象在堆区）

    // ----- 引用 -----
    int& ref = local;              // ref: 栈区（引用本质是指针常量，占用栈空间）

    // 释放堆内存
    delete p;                      // 释放p指向的堆内存
    delete[] arr2;                 // 释放arr2指向的堆数组
    delete pObj;                   // 释放pObj指向的堆对象
}

// ========== 主函数 ==========

int main(int argc, char* argv[]) { // argc: 栈区
                                   // argv: 栈区（指针数组，数组元素指向命令行参数字符串在栈/全局区）

    example(42);                   // 42作为实参传递给形参，进入栈区

    return 0;                      // 返回值通过寄存器或栈传递
}
