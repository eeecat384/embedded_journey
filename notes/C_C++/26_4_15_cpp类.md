# cpp类

## cpp类所解决的事情

在实际开发中，比如要记录一个玩家的坐标，速度，可以直接定义`int x, y, speed;`

但实际上不止一个玩家，如果还是用`int player0x, player0y, player0speed;`等等就过于臃肿

在函数调用这些变量时，比如`move(int x, int y, int speed)`

这不是一个很好的，紧密的结构，这时候就需要类来发挥作用

## 类的定义和使用

使用以下语法定义和使用Player类型：

```cpp
class Player
{
public: 
//类中的变量默认是私有的(private)，只有类中的函数可以访问，public可让类变为公有
    double x, y;
    double speed;
};

int main()
{
    Player player0; 
    //创建了一个变量player0(对象/实例)，变量类型为Player（类）
    player.x = 5;
    //使用.来访问对象
}
```

但是这是如果要写一个move函数，还是需要传入&player0，xa，ya三个参数，在move中使用点运算符。这仍然可以简化，我们可以直接在类中定义函数


```cpp
class Player
{
public: 
    double x, y;
    double speed;
    void move(int xa, int ya){
        x += xa
        y += ya
        //直接使用x，y，不用点运算符
    }
};

int main()
{
    Player player0; 
    player0.move(1, 2);
    //直接传入移动多少
}
```

## 类和结构体的区别

实际上，struct就是class的语法糖，是一个默认公共的class，其在C语言中可以使用，但class在C语言中并不兼容，无法使用

实际情况就是如此，他们在编译后的汇编代码层面是一致的，仅仅是默认公开或非公开和是否兼容C的区别

## 写一个cpp类：以日志为例子

>这不是一个很好的例子，实际开发中不会这么写，但这是一个很简单的例子来说明类怎么写

在一个用作日志功能的类中，我们想要将信息打印到屏幕上，我们可以将信息的等级分成三种：<br>
错误（error），警告（warning），信息（information）

于是我们定义三个变量LogLevelError/Warning/Info来增加程序可读性。有些时候，并不需要所有信息都打印出来，我们可能只需要打印警告和错误，就使用

```cpp
#include<iostream>
using namespace std;

class Log {
public:
	int LogLevelError = 0;
	int LogLevelWarning = 1;
	int LogLevelInfo = 2;
//为了增加可读性，在前面定义变量，后面定义函数

private:
	int m_logLevel = LogLevelInfo;
//定义m_loglevel来记录当前信息等级，默认为Info
//m_前缀表明其为私有变量，并非语法，而是区分公共和私有和增加可读性

public:
	void SetLevel(int level) {
		m_logLevel = level;
	}
//定义SetLevel函数来随时设置当前信息等级

	void Error(const char* message) {
		if (m_logLevel >= LogLevelError)
			cout << "[ERROR]" << message << endl;
	}

	void Warn(const char* message) {
		if (m_logLevel >= LogLevelWarning)
			cout << "[WARNING]" << message << endl;
	}

	void Info(const char* message) {
		if (m_logLevel >= LogLevelInfo)
			cout << "[INFO]" << message << endl;
	}
//定义三个函数来打印错误，警告，信息
};

int main()
{
	Log log;
	log.SetLevel(log.LogLevelWarning);
	log.Error("This is an error message.");
	log.Warn("This is a warning message.");
	log.Info("This is a info message.");
//在这里，由于LogLevel为Warning，故只有Error和Warn打印了出来
}
```
